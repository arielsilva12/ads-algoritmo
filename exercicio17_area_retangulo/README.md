# Exercício – Cálculo da Área de um Retângulo

Neste exercício, o programa lê o valor da base e da altura de um retângulo informado pelo usuário e calcula a sua área utilizando a fórmula: área = base * altura.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor da base do retângulo
2. O usuário informa o valor da altura do retângulo
3. O programa calcula a área utilizando a fórmula: área = base * altura
4. O resultado da área é exibido na tela
