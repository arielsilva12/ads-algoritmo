# Exercício – Cálculo do Volume de uma Esfera

Neste exercício, o programa lê o valor do raio de uma esfera informado pelo usuário e calcula o seu volume utilizando a fórmula: v = (4 * π * r³) / 3.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação, divisão e exponenciação)
- Uso de variáveis
- Constantes matemáticas (π)
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor do raio da esfera
2. O programa utiliza o valor de π (pi) para o cálculo
3. O programa calcula o volume utilizando a fórmula: v = (4 * π * r³) / 3
4. O volume da esfera é exibido na tela
