# Exercício – Gasto de Dinheiro de um Fumante

Neste exercício, o programa calcula a quantidade total de dinheiro gasta por um fumante ao longo de um determinado período, com base no número de anos que ele fuma, na quantidade de cigarros consumidos por dia e no preço de uma carteira de cigarros.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int` e `float`)
- Operações matemáticas básicas
- Uso de variáveis
- Saída de dados com `print`
- Cálculo proporcional

## Lógica do Exercício
1. O usuário informa quantos anos o fumante fuma
2. O usuário informa quantos cigarros são fumados por dia
3. O usuário informa o preço de uma carteira de cigarros (20 cigarros)
4. O programa calcula o preço unitário de cada cigarro
5. O programa calcula o gasto diário com cigarros
6. O programa multiplica o gasto diário pelo total de dias fumados (anos × 365)
7. O valor total gasto é exibido na tela
