# Exercício – Conversão de Dólar para Real

Neste exercício, o programa lê a cotação do dólar e um valor em dólar informado pelo usuário, calcula e exibe o valor equivalente em reais (R$).

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício

1. O usuário informa o valor da cotação do dólar
2. O usuário informa um valor em dólar
3. O programa calcula o valor em reais multiplicando o valor em dólar pela cotação
4. O resultado é exibido na tela em reais (R$)
