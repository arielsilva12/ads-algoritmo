# Exercício – Cálculo do Comprimento de uma Circunferência

Neste exercício, o programa lê o valor do raio de uma circunferência informado pelo usuário e calcula o seu comprimento utilizando a fórmula: c = 2 * π * r.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação)
- Uso de variáveis
- Constantes matemáticas (π)
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa o valor do raio da circunferência
2. O programa utiliza o valor de π (pi) para o cálculo
3. O programa calcula o comprimento utilizando a fórmula: c = 2 * π * r
4. O comprimento da circunferência é exibido na tela
