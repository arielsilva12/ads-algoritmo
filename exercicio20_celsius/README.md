# Exercício – Conversão de Temperatura para Escala Fahrenheit

Neste exercício, o programa lê uma temperatura informada em graus Celsius e realiza a conversão para a escala Fahrenheit utilizando a fórmula: F = (C × 9 / 5) + 32.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`float`)
- Operações matemáticas básicas (multiplicação, divisão e soma)
- Uso de variáveis
- Saída de dados com `print`
- Formatação de saída com f-string

## Lógica do Exercício
1. O usuário informa a temperatura em graus Celsius
2. O programa aplica a fórmula de conversão para Fahrenheit
3. O valor convertido em Fahrenheit é exibido na tela
