# Exercício – Cálculo de Expressão Matemática

Neste exercício, o programa lê três números inteiros positivos (A, B e C) e calcula o valor da expressão D, onde D = (R + S) / 2, sendo R = (A + B)² e S = (B + C)².

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas
- Uso do operador de potência (`**`)
- Uso de variáveis
- Saída de dados com `print`

## Lógica do Exercício
1. O usuário informa os valores de A, B e C
2. O programa calcula R elevando ao quadrado a soma de A e B
3. O programa calcula S elevando ao quadrado a soma de B e C
4. O programa calcula D utilizando a média de R e S
5. O resultado final é exibido na tela
