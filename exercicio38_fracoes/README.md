# Exercício – Soma de Duas Frações

Neste exercício, o programa lê duas frações (numerador e denominador) e calcula a soma entre elas, exibindo o resultado final em forma de fração.

## Conceitos Utilizados
- Entrada de dados com `input()`
- Conversão de tipos (`int`)
- Operações matemáticas
- Uso de variáveis
- Saída de dados com `print`
- Representação de frações

## Lógica do Exercício
1. O usuário informa o numerador e o denominador da primeira fração
2. O usuário informa o numerador e o denominador da segunda fração
3. O programa calcula o numerador da fração resultante utilizando a regra da soma de frações
4. O programa calcula o denominador da fração resultante
5. O resultado da soma é exibido no formato de fração
